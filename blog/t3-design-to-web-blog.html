What a responsive site is, and why responsiveness is important?

A website where its display and layout will tranform according to the screen size of different devices is a responsive site. The resposiveness is important to bring out the best user experiences for different devices. It also allow easy code maintenance because Web Developers no longer need to write separate HTML and CSS files to cater for different screen sizes

What mobile first design is, and why it's important?

Mobile First Design is a concept where a website is designed to look its best on mobile devices. And then gradually add addition consideration for enhancement to devices with bigger screen size. This is contrary to the the old approach of Desktop First Design. In Desktop First Design, a website is designed to look its best at Desktop devices, and then a lot of elements have to be removed or compromised to allow a webpage to fit into small screen size in mobile devices. At present day, the number of people accessing websites using mobile devices are way more than people accessing websites using desktop devices. So it is a high priority to design a webpage to look its best in mobile devices.

What frameworks are, and their pros and cons?

Frameworks are various set of library codes that have been written, tested, and ready to be used for development. Using framework, such as Skeleton or Bootstrap CSS framework, a developer can quickly reuse the code offered by the framework to quickly develop a webpage. Although convenient, framework itself will has limitation where modification is difficult. The webpage will also be affected by the bugs or security vulnerabilities of a framework.

What a wireframe is and why we use it?

A wireframe is a schematic or blueprint for a webpage. It is used to plan the overall look and feel without worrying too much about the details such as colors and fonts.
