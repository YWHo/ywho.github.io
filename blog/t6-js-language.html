parenthesis: ()

There are many ways to look at parenthesis:

  • grouping operator: Parenthesis can be used as an operator to group mathematical operations 
    a = 2 * (7 + b)
  • containing operator: Parenthesis can be used as a container to enclose an expression 
    if (a > b) {
    // other codes
}
  • function declaration (without or without function name): when a word 'function' is followed by '()', it indicates a function declaration. 
    function () {
    // other codes
}

function (param1, param2) {
    // other codes
}

function doSomething() {
    // other codes
}

function doMoreThing(param1, param2) {
    // other codes
}
  • invoking a function: Parenthesis at the back of a function name is invoking a function 
    doSomething();

doMoreThing("thing1", "thing2");
  • function expression: Parenthesis enclosing a function declaration is called 'function expression' 
    (function () {
    // other codes
})();

(function () {
    // other codes
}());

(function doSomething() {
    // other codes
}());
  • arrow notation: Parenthesis followed by an arrow '=>', is a shorter syntax for function expression 
    () => {
    // other codes
}

(param1, param2) => {
    // other codes
}

brackets: []

Brackets are mostly for operations related to Arrays

  • Array declaration: Brackets with contents after an assignment statement (=) is to create for array creation. 
    var fruits = ["Apple", "Orange", "Banana"];

var cars = [
        "Mazda",
        "Holden",
        "Toyota",
        "Nissan",
        "Honda"
    ];

var emptiness = [];
  • Accessing Array elements: Brackets with a numeric value after an Array name is to access an element in an array. The numeric value inside the brackets is the index of array elements. 
    fruits[1]
                                
cars[2]
  • Accessing Object properties: Brackets with a string value after an Object name is to access an element in an object. 
    person["name"]

car["door"]

braces {}

Braces are for blocks and objects

  • Indicating a block: Braces are to indicate a block, in the case of if-then-else, switch-statement, looping, or functions 
    if (...) {
    //
    // content inside a block
    //
}

switch(...) {
    //
    //content inside a block
    //
}

while (...) {
    //
    // content inside a block
    //
}

for (;;) {
    //
    // content inside a block
    //
}

function doSomething() {
    //
    // content inside a block
    //
}

var xyz = function() {
    //
    // content inside a block
    //
}
  • Object literals: Braces after an assignment statement (=) with the content name (or key) value pairs is for object creation. 
    var person = {
    name : "Bob",
    age : 35,
    gender : "Male",
    occupation : "Bank security"
}

var car = {
    type : 'Toyota',
    model : 'Prius C',
    make : '2017',
    price : '$29990',
};

single quotes: ''

Single quotes are for string creation 

var say = 'She sells seashells by the seashore'

var name = 'Tony Stark'

double quotes: ""

Double quotes are also for string creation 

var say = "She sells seashells by the seashore"
                            
var name = "Tony Stark"
