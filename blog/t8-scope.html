What 'scope' is, and how it works in JavaScript.

'Scope' is a term used in programming to mean the visibility of a variable or function. There are many type of scopes in Javascript:

  • Global scope: Any variable declared outside of functions inside a file is globally visible to all the functions or expression within the file. For example: 
    var temperature = 32

function myFunction() {
    console.log("Current temperature is: " + 32 + " degree celsius");
    // output: Current temperature is: 32 degree celsius
}
  • Local scope: Any variable declared inside a function will not be visible outside of the function. For example: 
    function myFunction() {
    var innerValue = 6;
          
    console.log("Within function: " + innerValue);
    // output: Within function: 6
}

console.log("Outside function: " + innerValue);
// Browser will report "ReferenceError: innerValue is not defined"
  • Auto global scope: If there is an assignment expression on a variable that is never declared, the variable is treated as a global variable and visible globally. For example: 
    myFunction();
console.log("The car is: " + car);
// Output: The car is: mazda
                                
function myFunction() {
    car = "mazda;
}
