The story of my problem solving

This home page (ywho.github.io) has the problem of code duplication at side navigation bar. Originally, every single post will need to have its side navigation updated individually. As the number of posts increased, the workload increase also. To solve the problem of code duplication, I tried to do some googling to see if I can include html file in some way. I just need to use 'w3-include-html' attribute and a link to 'w3.js' file in the script tag.

Although the problem was solved, I faced new follow up problems. Anyway, let just focus on the first problem that I mentioned above. After solving the first problem, I was feeling really happy, because I can immediately see the effect of saving time in updating the side navigation bar.

From the experience, I learnt that there is always a solution at some place, because someone might have the same problem before me.

About .map() in Javascript Array

The .map() method will internally loop through every element in an array, then pass each element to a function to be processed. Each processed result will be saved into a new array. At the end of the looping, the new array will be returned as the result of the operation.

About .filter() in Javascript Array

The .filter() method will internally loop though every element in an array, then test every element with the test condition provided. Any element that passed the test condition will be saved into a new array. At the end of the looping, the new array with those elements that passed the test condition, will be returned.

About .reduce() in Javascript Array

The .reduce() method is a shortcut way to accumulate every element inside the array to a single result. First, it has a default Total that starts at 0. Then it will internally loop through every element from left to right. Each element will do mathematic operation with Total, and save the result back to Total. The new Total will be used for operation with the next element. The final result will be the accumulated results of the mathematic operation upon every element. Regarding the mathematic operation, it can be any operation as long as the operation involves both Total and an array element.
